The History of lolnet

We started out as a private Minecraft server for some friends on a Pentium 4 with 2GB RAM and 160GB HDD, it was located in Christchurch on an ADSL line and started probably about December 2010.

In January 2011 I (CptWin) decided to open up the Minecraft Server and make it public as there was a need for a server based in New Zealand. I called it lolnet as that was the name of our wireless network at the time and it sounded cool and matched my favourite lolface logo.

We started out with the domain http://lolnet.dyndns.biz which is a free domain with dyndns.com and as word got out the server got busier and busier with a peak of about 1000 unique players in around 6-8 months.

We had a refresh in January 2012 and the server was moved from Christchurch to Auckland and new hardware was bought due to the generous Donations of a large number of our players!

The server was upgraded to a Core i3 3.1GHz with 8GB RAM and 3x1TB HDD in RAID 0 and we finally got a .co.nz and so lolnet.co.nz was created. At that point we we're serving around 400 players on the forums and 1200 players on our Minecraft Server.

Around May 2012 because of this increase in size we eventually moved the forums and Minecraft server into a dedicated cloud server which gives us the ability to scale the server up in power to host as many players as we want, which as of 5/09/2012 is 4740 Minecraft players and 2181 forum members!

August 2012 saw a wild Lord_Mega appear he hosted a Minecraft Tekkit Server on his EliteServer! (Core 2 Duo 1.8Ghz, 5GB RAM, 160GB HDD, ADSL2+)

January 2013 Dezil_NZ opened up a Feed the Beast Server.

31/01/2013 lolnet Minecraft is hosting for 10,835 unique players and 5028 forum members!

06/03/2013 lolnet Minecraft is hosting for 13,388 unique players and 6309 forum members!

27/07/2013 lolnet Minecraft is hosting for 19,106 unique players and 6924 forum members!

16/09/2015 lolnet Minecraft is hosting for 25,972 forum members!

lolnet and it's servers have always been free to play, and will always continue to be free to play!
